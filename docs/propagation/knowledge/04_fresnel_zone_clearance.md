# 04 菲涅尔区净空

## 1. 核心理论

即使几何 LOS 成立，若第一菲涅尔区被侵入，也会产生附加损耗。  
因此“可视距”不等于“高质量链路”。

## 2. 关键公式

第一菲涅尔区半径：

```text
r1 = sqrt(λ * d1 * d2 / (d1 + d2))
```

工程规则：

- 净空 < 60%：通常风险上升
- 侵入严重：增加 3~10dB（按经验/校准）

## 3. 在系统中的用途

### 3.1 作用点

- 作为 LOS 场景的质量修正项。
- 支持“看起来通视但链路差”的解释。

### 3.2 输入输出

输入：路径几何、频率、障碍与直射线间隙。  
输出：净空比例、附加损耗 `Lfresnel`。

## 4. 用在哪一方

- 数据层：提供剖面
- 传播计算层：附加损耗计算
- 网络仿真层：间接影响 SINR
- 优化决策层：用于筛除边缘链路
- UI/可视化层：剖面显示菲涅尔区
- 校准学习层：可拟合阈值/惩罚系数

## 5. 实现注意

- 不同频段的菲涅尔半径不同，频率越低半径越大。
- 切勿与几何遮挡重复计损。

## 6. 验收要点

- 净空比例下降时 `Lfresnel` 上升。
- 剖面图中菲涅尔区渲染与计算一致。
