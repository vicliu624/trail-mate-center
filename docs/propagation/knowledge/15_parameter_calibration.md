# 15 参数校准与误差闭环

## 1. 核心理论

模型初值来自经验，必须通过实测数据校准。  
目标是最小化预测与观测差异，并形成可追溯版本。

## 2. 典型目标函数

- MAE 最小化
- RMSE 最小化
- 分场景分层误差最小化

## 3. 在系统中的用途

### 3.1 作用点

- 拟合 `α_class`、`σ`、反射参数。
- 生成“校准前后误差对比”。

### 3.2 输入输出

输入：实测 RSSI、GPS 轨迹、节点参数。  
输出：新参数集、误差报告、参数版本号。

## 4. 用在哪一方

- 数据层：存储实测数据
- 传播计算层：应用新参数
- 网络仿真层：间接受益
- 优化决策层：提高评分可信度
- UI/可视化层：Calibration 报告展示
- 校准学习层：核心

## 5. 实现注意

- 训练/验证数据要分开，避免过拟合。
- 参数变更要版本化并可回滚。

## 6. 验收要点

- 校准后 MAE/RMSE 有可量化下降。
- 新参数在验证集也有效。
