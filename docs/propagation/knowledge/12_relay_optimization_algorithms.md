# 12 中继部署优化算法

## 1. 核心理论

把中继选址建模为多目标优化问题：覆盖、可靠性、干扰、成本之间的权衡。

典型评分：

```text
Score = w1*Coverage + w2*Reliability - w3*BlindArea - w4*Interference - w5*Cost
```

## 2. 算法分级

- 贪心：实现简单，速度快，适合第一版
- 遗传算法：全局搜索能力更强
- 模拟退火：可跳出局部最优，适合科研模式

## 3. 在系统中的用途

### 3.1 作用点

- 输出 TopN 推荐方案。
- 支持方案对比与一键回放。

### 3.2 输入输出

输入：候选点集、评分权重、约束条件。  
输出：方案列表、各项得分分解。

## 4. 用在哪一方

- 数据层：提供候选点与约束数据
- 传播计算层：提供每个点位的覆盖贡献
- 网络仿真层：提供干扰/容量约束
- 优化决策层：核心
- UI/可视化层：TopN 展示与对比
- 校准学习层：可学习权重策略

## 5. 实现注意

- 必须输出“为何推荐”而不只是编号。
- 约束条件应可配置（数量、间距、可达性）。

## 6. 验收要点

- Top1 至少在一个核心指标上优于基线方案。
- 方案得分与可视化结果一致。
