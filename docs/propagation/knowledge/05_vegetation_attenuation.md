# 05 植被穿越损耗模型

## 1. 核心理论

植被会对 LoRa 链路产生显著附加衰减，尤其在密林、灌木密集区。  
常见工程模型按路径穿越长度累加：

```text
Lveg = Σ(α_class × d_class)
```

- `α_class`：地类衰减系数（dB/m）
- `d_class`：该地类穿越长度（m）

## 2. 典型系数（初值）

- 稀疏林：0.2~0.4 dB/m
- 密林：0.5~1.5 dB/m
- 草地：通常显著低于林地

## 3. 在系统中的用途

### 3.1 作用点

- 解释“地形通视但信号仍弱”的典型场景。
- 支撑林区任务（搜救、巡护）的现实预测。

### 3.2 输入输出

输入：土地覆盖栅格、路径采样点、分类映射、`α_class`。  
输出：`VegetationLoss`、分类贡献分解。

## 4. 用在哪一方

- 数据层：土地覆盖数据接入与映射
- 传播计算层：植被损耗累加
- 网络仿真层：影响链路强度底图
- 优化决策层：影响候选点评分
- UI/可视化层：Loss Breakdown 的 Vegetation
- 校准学习层：拟合 `α_class`

## 5. 实现注意

- 城市地物建议单独建模，不并入植被项。
- 分类编码需统一，避免跨数据源语义不一致。

## 6. 验收要点

- 路径穿越植被长度越长，损耗越大。
- 分类贡献可被 UI 明确展示。
