# 11 基于 DEM 的山脊候选提取

## 1. 核心理论

山脊点通常具备更好的视距条件，是中继候选点的重要来源。  
可通过局部极值、坡度、曲率综合提取。

## 2. 常用方法

- 局部高程极大值检测
- 坡度阈值过滤（剔除不可部署陡坡）
- 曲率分析区分脊线与谷线

## 3. 在系统中的用途

### 3.1 作用点

- 自动生成中继候选集合，减少人工试点成本。

### 3.2 输入输出

输入：DEM、坡度阈值、最小点间距。  
输出：候选点列表（含评分先验）。

## 4. 用在哪一方

- 数据层：DEM 派生层生成
- 传播计算层：间接使用
- 网络仿真层：间接使用
- 优化决策层：核心候选生成
- UI/可视化层：候选点高亮与筛选
- 校准学习层：可根据历史部署效果调整规则

## 5. 实现注意

- 候选点过多会拖慢优化，需聚类降采样。
- 需结合禁入区、道路可达性等约束。

## 6. 验收要点

- 候选点主要分布在脊线附近。
- 筛选后数量满足优化计算规模要求。
