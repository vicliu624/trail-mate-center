# 09 SINR 与 LoRa 干扰机制

## 1. 核心理论

多节点网络中，接收质量由 `SINR` 决定：

```text
SINR = S / (I + N)
```

- `S`：目标信号功率
- `I`：干扰信号总功率
- `N`：噪声功率

LoRa 特性：不同 SF 具有准正交性，但并非绝对无干扰。

## 2. 在系统中的用途

### 2.1 作用点

- 生成干扰热力图。
- 评估冲突率与容量风险。

### 2.2 输入输出

输入：节点集合、发包负载、传播场、SF 配置。  
输出：`SINR` 栅格、冲突概率、受影响节点列表。

## 3. 用在哪一方

- 数据层：不直接使用
- 传播计算层：提供 `S/I` 基值
- 网络仿真层：核心
- 优化决策层：干扰惩罚项
- UI/可视化层：Interference Analysis 模式
- 校准学习层：可用实测冲突日志校验

## 4. 实现注意

- 同频同 SF 冲突应作为高优先级处理。
- 不同 SF 可按捕获矩阵近似，而不是直接忽略干扰。

## 5. 验收要点

- 增加节点密度后干扰热图应增强。
- 高负载时冲突率明显上升。
