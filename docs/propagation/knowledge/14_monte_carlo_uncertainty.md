# 14 蒙特卡洛不确定性仿真

## 1. 核心理论

通过对关键参数做随机扰动并重复仿真，评估结果稳定性和置信区间。

## 2. 典型扰动项

- 植被密度或 `α_class`
- 阴影衰落 `σ`
- 地表反射参数

## 3. 在系统中的用途

### 3.1 作用点

- 输出“方案鲁棒性”，避免只看单次最优。
- 给出 95% 可靠性与置信区间。

### 3.2 输入输出

输入：基础参数、扰动分布、迭代次数 `N`。  
输出：指标分布、置信区间、稳定性评分。

## 4. 用在哪一方

- 数据层：不直接使用
- 传播计算层：批量重复计算
- 网络仿真层：可做概率化容量评估
- 优化决策层：鲁棒优化依据
- UI/可视化层：Advanced Modeling 面板
- 校准学习层：可验证参数不确定性影响

## 5. 实现注意

- 计算成本高，需要并行与缓存。
- 迭代次数应可配置（快速/标准/高精）。

## 6. 验收要点

- 同一场景可输出稳定性排序。
- 迭代次数增加后统计量收敛。
