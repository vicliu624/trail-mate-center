# 01 自由空间损耗与链路预算

## 1. 核心理论

链路预算是无线规划的基础：把发射端“预算”到接收端，得到接收功率 `Pr`。

最基础公式：

```text
Pr = Pt + Gt + Gr - Ltotal
```

其中 `Ltotal` 至少包含自由空间损耗 `FSPL`，其余损耗按模型叠加。

## 2. 关键公式

```text
FSPL(dB) = 32.44 + 20log10(f_MHz) + 20log10(d_km)
```

- `f_MHz`：频率（MHz）
- `d_km`：路径距离（km）

## 3. 在系统中的用途

### 3.1 作用点

- 作为所有覆盖计算、单点分析、方案比较的“基线损耗”。
- 作为右侧 `Loss Breakdown` 卡片的第一项。

### 3.2 输入输出

输入：频率、节点位置、天线参数。  
输出：`FSPL`、基础 `Pr`、基础 `Margin`。

## 4. 用在哪一方

- 数据层：不使用
- 传播计算层：核心公式
- 网络仿真层：作为 `S` 与 `I` 的底层输入
- 优化决策层：评分计算依赖
- UI/可视化层：展示 `FSPL` 与 `Pr`
- 校准学习层：作为误差分解基线

## 5. 实现注意

- 距离过短时避免 `log10(0)`，设置最小距离阈值（如 1m）。
- 统一单位：频率 MHz、距离 km，避免单位错配。
- 公式是自由空间假设，山地环境需叠加地形/植被/阴影项。

## 6. 验收要点

- 单元测试覆盖多个频段（433/470/868/915MHz）。
- 与手工计算结果误差 < 0.01dB。
