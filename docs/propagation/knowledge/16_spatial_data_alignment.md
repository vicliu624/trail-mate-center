# 16 空间数据对齐与重采样

## 1. 核心理论

传播模型对空间精度高度敏感。  
若 DEM、土地覆盖、路径采样坐标不一致，会导致系统性误差。

## 2. 关键概念

- 坐标参考系（CRS）统一
- 像元对齐（grid alignment）
- 重采样策略（nearest/bilinear/cubic）
- NoData 处理

## 3. 在系统中的用途

### 3.1 作用点

- 保障 LOS、植被累计、候选点提取结果可信。
- 保障 2D/3D 叠加图层不偏移。

### 3.2 输入输出

输入：原始多源栅格与目标网格规范。  
输出：统一网格数据、对齐校验报告。

## 4. 用在哪一方

- 数据层：核心
- 传播计算层：依赖一致输入
- 网络仿真层：依赖一致输入
- 优化决策层：依赖候选点精度
- UI/可视化层：地图叠加一致性
- 校准学习层：减少假误差

## 5. 实现注意

- 分类数据重采样优先 nearest，避免类别污染。
- 连续变量（高程）可用 bilinear/cubic。
- 必须落地自动化对齐校验。

## 6. 验收要点

- 同一点在各图层采样坐标一致。
- 叠加渲染无明显偏移。
