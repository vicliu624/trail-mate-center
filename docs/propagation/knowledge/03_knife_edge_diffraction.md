# 03 单刀锋绕射模型

## 1. 核心理论

当 LOS 被地形阻挡时，单刀锋模型用于估算主要障碍导致的附加损耗 `Ld`。

## 2. 关键公式

```text
v = h * sqrt((2 / λ) * (1/d1 + 1/d2))
```

```text
if v > -0.7:
  Ld = 6.9 + 20log10(sqrt((v-0.1)^2 + 1) + v - 0.1)
else:
  Ld = 0
```

- `h`：障碍超出直射线的高度
- `d1/d2`：障碍到 Tx/Rx 距离
- `λ`：波长

## 3. 在系统中的用途

### 3.1 作用点

- NLOS 路径损耗主项。
- 右侧 `Loss Breakdown` 的 `Diffraction` 字段。

### 3.2 输入输出

输入：主障碍点、频率、路径几何。  
输出：`v`、`Ld`。

## 4. 用在哪一方

- 数据层：提供剖面地形
- 传播计算层：核心模型
- 网络仿真层：影响 `S/I` 基值
- 优化决策层：中继点选择时重要指标
- UI/可视化层：可视化绕射点与绕射区
- 校准学习层：可引入校正系数

## 5. 实现注意

- 只取“主刀锋”是近似，复杂地形可能低估损耗。
- 高频时对遮挡更敏感，`Ld` 变化更明显。

## 6. 验收要点

- 障碍高度增大时 `Ld` 单调上升。
- `v <= -0.7` 时 `Ld` 接近 0。
