# 06 对数正态阴影衰落

## 1. 核心理论

阴影衰落表示大尺度慢衰落随机性，常建模为对数正态：

```text
Pr ~ N(μ, σ)
```

其中 `μ` 是平均接收功率，`σ` 是标准差（dB）。

## 2. 典型参数

- 山地场景 `σ` 常取 6~10 dB（初值）。

## 3. 在系统中的用途

### 3.1 作用点

- 从“确定性覆盖”扩展到“概率覆盖”。
- 输出 95% 与 80% 可靠性覆盖图。

### 3.2 输入输出

输入：平均 `Pr`、灵敏度阈值、`σ`。  
输出：`P(Pr > Sensitivity)`。

## 4. 用在哪一方

- 数据层：不直接使用
- 传播计算层：概率覆盖核心
- 网络仿真层：可用于概率化容量评估
- 优化决策层：评分中的可靠性项
- UI/可视化层：Coverage Probability 卡片
- 校准学习层：通过实测拟合 `σ`

## 5. 实现注意

- `σ` 不应全域固定，可按地类/地形分区。
- 结果需要和确定性覆盖并行展示，避免误读。

## 6. 验收要点

- `σ` 增大时，边缘区域可靠性下降。
- 95% 覆盖区应严格小于 80% 覆盖区。
