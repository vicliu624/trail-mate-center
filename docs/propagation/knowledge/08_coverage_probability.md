# 08 覆盖概率计算

## 1. 核心理论

覆盖不应仅是“可达/不可达”二元判断，而应输出概率：

```text
Pcov = P(Pr > Sensitivity)
```

在引入阴影衰落后可得到不同置信度下的覆盖范围。

## 2. 在系统中的用途

### 2.1 作用点

- 输出 `95%` 与 `80%` 可靠性图层。
- 作为优化目标中的可靠性项。

### 2.2 输入输出

输入：平均功率 `μ`、标准差 `σ`、阈值。  
输出：`Pcov` 栅格。

## 3. 用在哪一方

- 数据层：不直接使用
- 传播计算层：概率化后处理
- 网络仿真层：可用于业务成功概率估算
- 优化决策层：覆盖概率评分
- UI/可视化层：Coverage Probability 图例与统计
- 校准学习层：评估概率模型是否过于乐观/保守

## 4. 实现注意

- 需明确概率是“时空统计意义”，不是瞬时保证。
- 输出应包含阈值说明（例如 95% 可靠性）。

## 5. 验收要点

- 可切换显示平均覆盖与概率覆盖。
- 同一区域不同阈值下结果变化符合预期。
